# Reflection

## 1. Vanilla JS khó ở đâu?

Phải tự thao tác DOM bằng:
- getElementById
- innerHTML
- appendChild

Khi ứng dụng lớn sẽ rất khó quản lý.

---

## 2. React dễ hơn điểm nào?

React chỉ cần cập nhật state.
UI sẽ tự động render lại.

Không cần thao tác DOM trực tiếp.

---

## 3. Imperative vs Declarative là gì?

Vanilla JS:
- Imperative
- Phải chỉ từng bước sửa giao diện

React:
- Declarative
- Chỉ mô tả UI muốn hiển thị

---

## 4. Điều em thích ở React?

Code ngắn gọn hơn.
Dễ quản lý state và UI.
Component tái sử dụng tốt hơn.
